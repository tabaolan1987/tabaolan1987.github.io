Working With TestLink : How to Install TestLink?

Perquisite

  1. Download and install XAMPP application into your machine from link: Click here

  2. Download TestLink from link : Click Here

  3. Extract downloaded TestLink zip file and rename this to “TestLink”.

  4. Go to folder “\xampp\htdocs” and copy test link to it.

  5. Start Xampp application and start “Apache” and “MySql” Server, your window look like below screen:

Note : The default port for Apache is : 80 and Mysql is : 3306. You should check there port are not use in your computer.

Setup

  1. Open browser and go to url “localhost/phpmyadmin”

  2. Click on “DataBases” menu and create new data base like “testlink”

  3. Your created database listed in left side panel like as in below screen:

  1. Click on created database from left panel then Click on “More” >> “Privileges”.

  2. Click on “Add User”after that enter user details and click on “Check All” link.

  1. Click on “Go” button. Now your database created with database user.

  2. Navigate “http://localhost/testlink%E2%80%9D on browser.

  1. Click on “New installation” link.

  2. Click on check box “I agree to the terms set out in this license.” And click on “Continue” button. You should see below screen:

  1. You should see that “Checking if /var/testlink/logs/ directory exists” and “Checking if /var/testlink/upload_area/ directory exists ” status failed.

  2. You need to set path according your machine.

  3. Goto testlink package and open “config.inc.php” file, update path “$tlCfg->log_path” and “$g_repositoryPath” as below:

  1. Again refresh page and you should see that “Failed” status change with “OK”

  1. Click on “Continue” button. Enter your data base name into “Database name” input field. Enter root into “Database admin login” input filed.

  2. Enter data base user name and password into “TestLink DB login” and “TestLink DB password” field.

  3. Click on “Process TestLink Setup” button. You should see that test link “Installation was successful!” message display.

  1. Go to url “http://localhost/testlink%E2%80%9D. and enter “admin” into both “Login Name” and Password” field. Click on “Login” button.
