IceCream Framework Part 3 : Running an Example Test Case.

Running an example test case

Follow to there steps below :

1) Open IceCream folder.You will see the file ‘TESTEXAMPLE.xslx’.

3) Copy this file and paste to Folder TestCase then rename this file like you want.

4) Open this file you copied by Microsoft Excel.

5) In this file , I create my example test case is : Running in Browser Chrome , Open ‘http://facebook.com’ , wait for text-box password present in 5 second , then I type in text-box email , type in text-box password. Take a screenshot and close the browser.

5) First take a look in the top , you can able to set up browser you want to run this test case. The list provide you 3 browser : Firefox , IE , Chrome. Make sure that 3 browser is install in your laptop. Or you can download Firefox browser here , Chrome browser here. In this case , I choose my test case running in Chrome. You can set IE ,CHROME ,FIREFOX like you want.

6) Now take a look in row 3. I use the action ‘OpenUrl’ with value is ‘http://facebook.com’ , It mean that I will let browser access to url facebook.com and wait for the facebook.com load success.You can change whatever URL you like in this Action Input Value.

7) Take a look in row 4 . I use the action ‘WaitElementPresent’ with the Action Type Input is 5.It mean that I will wait 5 second for the Element that I find BY ID is pass present in this page.

8) You can see row 4- 8 some action , but I think you need to see the How to use Action in Excel

9) Now save and close this file.

10) Back to folder IceCream , double click in file ‘run.cmd’ and see the automation test process.

11) After automation test process success , check Report in folder Report and check the screenshot in folder Screenshot.

12) Now you can create your own test case by copy file ‘TESTEXAMPLE.xslx’ to the test case folder, rename this file , open this file , choose the browser , and delete row 3 to 8 and define your action in this excel file.We recommend you to take a look in How to use Action in Excel . If you have any trouble to set up and run your test case , leave your comment below.
