IceCream Framework Part 4 :Function available in excel file and how to use!

Jun 10th, 2014
  • IceCream-Framework
Comments

Version Beta 0.1

Function Browser

1) Define : choose your browser you want to run your test case.

2) Available : Chrome , FireFox , IE.

3) Required : All browser must be install in your laptop or computer.

Function OpenUrl

1) Define : open the url you want to test.It will wait for the browser load this page success!

2) Available : with any url that publish to internet or your local network.

3) Required : correct the format of URL such as : www.anyurl.com or http/https://anyurl.com. Must put value in column ‘ACTION INPUT VALUE’.

Function CloseBrowser

1) Define : close the browser.

2) Available : All browser that you choose in the top of excel file.

3) Required : don’t need to put any value in ‘ACTION INPUT VALUE’ , ‘FIND BY’, ‘FIND BY VALUE’

Function Type

1) Define : type in text-box that you want.Return false if element is not display or not is text box or text arena.

2) Available : all text-box that visible in website.

3) Required :

‘ACTION INPUT VALUE’ : your value you want to type in text-box!

‘FIND BY’ : Choose once of options : ‘ID’ , ‘TagName’ , ‘ClassName’ , ‘Css Selector’ , ‘Xpath’ , ‘Link text’ , ‘Partial Link Text’

‘FIND BY VALUE’ : give the value like xpath of this text-box if you choose Xpath option in ‘Find By’.

4) Example : Let’s take a look in this html source below :

The scenario is find the textbox and type in it value ‘example’ by using function type in excel.

I choose the demo to you 3 option is :

A)Find by ID :

‘ACTION INPUT VALUE’ : your value you want to type.

‘FIND BY’ : ID

‘FIND BY VALUE’ : textbox (this is the id of this input type)

B)Find by Class name :

‘ACTION INPUT VALUE’ : your value you want to type.

‘FIND BY’ : Class Name

‘FIND BY VALUE’ : example(this is the class name of this input type)

C)Find by Xpath :

‘ACTION INPUT VALUE’ : your value you want to type.

‘FIND BY’ : Xpath

‘FIND BY VALUE’ : //input[id=‘textbox’] (this is the xpath of this textbox)

Function Click

1) Define : Click into button , link,.. etc! Return false if element is not displayed.

2) Available : All element is visible in website.

3) Required :

‘FIND BY’ : Choose once of options : ‘ID’ , ‘TagName’ , ‘ClassName’ , ‘Css Selector’ , ‘Xpath’ , ‘Link text’ , ‘Partial Link Text’

‘FIND BY VALUE’ : give the value like xpath of this element you want to click if you choose Xpath option in ‘Find By’.

4) Example : take a look in html source below

The scenario is find the link and click in it.

I choose the demo to you 2 option is :

A) Find by Class name :

‘FIND BY’ : Class Name

‘FIND BY VALUE’ : link_example(this is the class name of this link)

B) Find by Link text :

‘FIND BY’ : Link text

‘FIND BY VALUE’ : Click here(this is the text in the link)

Function VerifyTitlePage

1) Define : compare the title page with the text user input.Return false if the title page not equal with the text user input.

2) Available : String for input.

3) Required : ‘ACTION INPUT VALUE’ : give your String to compare with the current title page.

Function VerifyText

1) Define : compare text inside element with text user input.Return false if they are not equals.

2) Available : String input.

3) Required :

‘ACTION INPUT VALUE’ : give your String to compare with the current text in element.

‘FIND BY’ : Choose once of options : ‘ID’ , ‘TagName’ , ‘ClassName’ , ‘Css Selector’ , ‘Xpath’ , ‘Link text’ , ‘Partial Link Text’

‘FIND BY VALUE’ : give the location of element like xpath, ID you want to compare.

Function SelectValueDropdown

1) Define : select the value of drop-down list via value user input.Return false if value is not existed in drop-down.

2) Available : String input.

3) Required :

‘ACTION INPUT VALUE’ : give your String Value you want selected in drop-down

‘FIND BY’ : Choose once of options : ‘ID’ , ‘TagName’ , ‘ClassName’ , ‘Css Selector’ , ‘Xpath’ , ‘Link text’ , ‘Partial Link Text’

‘FIND BY VALUE’ : give the location of drop-down like Xpath , or ID you want to select!

‘FIND BY’ : ID

Function SwitchToFrame

1) Define : switch the driver to iframe.

2) Available : All I-frame visible in website.

3) Required :

‘FIND BY’ : Choose once of options : ‘ID’ , ‘TagName’ , ‘ClassName’ , ‘Css Selector’ , ‘Xpath’ , ‘Link text’ , ‘Partial Link Text’

‘FIND BY VALUE’ : give the value to find the location of IFrame like Xpath , or ID you want to select!

Function SwitchToWindow

1) Define : if browser current have more than once tab , it use to switch to the another tab.

2) Available : All tabs are existed in current browser.

3) Required :

‘ACTION INPUT VALUE’ : name of this tab you want to come.

Function SwitchToDefault

1) Define : switch to the default tabs

2) Available : All tabs are existed in current browser.

3) Required : Just choose the function only , not required any parameters.

Function EnterAlertAcceptButton

1) Define : enter the accept button in Alert if alert present.

2) Available : all alert is present.

3) Required : just choose the function only , not required any parameters.

Function EnterAlertDismissButton

1) Define : enter the cancel button in Alert if alert present.Return false if alert is not present.

2) Available : all alert is present.

3) Required : just choose the function only , not required any parameters.

Function IsAlertPresent

1) Define : check if alert is present.Return false if alert is not present.

2) Available : all browser.

3) Required : just choose the function only , not required any parameters.

Function IsElementPresent

1) Define : check if element is present.Return false if element is not present.

2) Available : all element.

3) Required :

‘FIND BY’ : Choose once of options : ‘ID’ , ‘TagName’ , ‘ClassName’ , ‘Css Selector’ , ‘Xpath’ , ‘Link text’ , ‘Partial Link Text’

‘FIND BY VALUE’ : give the value to find the location of element like Xpath , or ID you want to check!

Function WaitElementPresent

1) Define : wait for element present in website with time out default is 10 second , you can set this timeout by enter in column ‘ACTION VALUE INPUT’.Return false if element not present.

2) Available : all element.

3) Required :

‘ACTION INPUT VALUE’ : time out you want to wait for this element.

‘FIND BY’ : Choose once of options : ‘ID’ , ‘TagName’ , ‘ClassName’ , ‘Css Selector’ , ‘Xpath’ , ‘Link text’ , ‘Partial Link Text’

‘FIND BY VALUE’ : give the value to find the location of element like Xpath , or ID you want to check!

Function WaitElementsInvisible

1) Define : wait for all elements invisible in the timeout.Return false if once of elements is not invisible.

2) Available : List elements.

3) Required :

‘ACTION INPUT VALUE’ : time out you want to wait for this element.

‘FIND BY’ : Choose once of options : ‘ID’ , ‘TagName’ , ‘ClassName’ , ‘Css Selector’ , ‘Xpath’ , ‘Link text’ , ‘Partial Link Text’

‘FIND BY VALUE’ : give the value to find the location of element like Xpath , or ID you want to check!

Function TakeScreenShot

1) Define : take a screen-shot in your current browser tabs and save it under folder ‘Screenshot’.

2) Available : all browsers.

3) Required : just choose only action and not required any parameters for this.
